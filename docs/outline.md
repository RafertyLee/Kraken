## Project Kraken 项目大纲

- kraken（优先级最高）
  
  - 前端通讯
  
  - 底层api
  
  - 中层，集成度高的模块
    
    - 举例：sas，rcs自动控制
  
  - 高层，集成度很高的模块
    
    - 举例：自动发射（经典，pegas），自动对接，自动降落
  
  - 对外接口
    
    - 为不精通编程的用户提供傻瓜式操作（甚至积木式编程）

- kraken_gui（计划中）
  
  - 提供数据可视化
  
  - 提供用户自定义控件

- kraken_ca（可选）
  
  - 并行计算加速

- kraken_krpc（计划中）
  
  - 修改krpc，更高级的控制
